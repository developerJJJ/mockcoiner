*coin.py로 실행해 주세요.
*그간 만들었던 알고리즘들은 다른 py 파일에 들어 있습니다.


1.LinearClassfierDialog.py
평균회귀 예측 모델 py프로그램

2.LSTM.py
LSTM 딥러닝 가격 예측 모델 ( 속도 문제로 ★exe 파일에 미포함된 모듈 )
(실행시 딥러닝을 시작한 후에 UI를 호출 함으로 기다려 주세요.)

3.ror.py
변동성 돌파 알고리즘 모델 및 수수료 모델(함수 형태로 제공 됨으로 하단에 있는 함수 호출 부분의 주석을 제거 해야 함)

4.Real_trade.py
실시간 거래 모델(Coin.exe 사용 권장,사용시 상단 주석 제거)

5.Coin.py
코인 GUI 기반 실행 파일( 1,3파일을 import 하기 때문에 실행이 잘 안될시 같은 폴더에 넣어주어야 함 )

*LSTM과 ror.py는 exe파일에 없는 코드가 수록되어 있습니다.
다른 파일의 경우 exe파일에 모두 포함된 기능들입니다.

*그 외의 오류는 책자의 READ ME 부분을 확인해 주세요.
